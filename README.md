## Getting Started

Install packages:

```bash
npm i 
# or 
yarn install
```

Run json-server (Required)

```bash
# on localhost:4003
npm run json-server
# or
yarn run json-server
```

Run app

```bash
npm run cypress
# or
yarn run cypress
```

Run tests

```bash
npm run dev
# or
yarn run dev
```
